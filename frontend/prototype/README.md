Open directly:

```text
C:\src\appfactory-video-ia\frontend\prototype\index.html
```

Recommended local preview:

```bash
cd C:\src\appfactory-video-ia\frontend\prototype
python -m http.server 4173
```

Then open:

```text
http://localhost:4173/
```
